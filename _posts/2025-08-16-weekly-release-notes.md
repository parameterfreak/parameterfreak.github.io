# PaperOps 주간 릴리즈 노트
## 2025년 8월 10일 - 8월 16일

### 주요 개발 내용 요약

이번 주는 사용자 관리, UI/UX 개선, 데이터베이스 마이그레이션, 그리고 AI 통합 기반 구축에 중점을 둔 집중적인 개발 기간이었습니다.

### 1. 관리자 사용자 관리 시스템 (Admin User Management)

- **주요 기능**:

  - 사용자 검색 및 필터링
  - 역할 기반 접근 제어 (USER/ADMIN)

- **주요 기능**:
  - 프로젝트 메타데이터 관리
  - 프로젝트 상세 정보 모달

- **커밋**: `73ed657`

  - 향상된 성능 및 확장성

### 4. 종합적인 논문 삭제 기능

  - 데이터베이스 제약 조건 위반 해결

- **커밋**: `b3eb662`, `71401a4`, `03b5685`
