## H-MAS v0.4.0 Developer Preview

> **"실제 클러스터에서 모델이 돌아간다"**

xxxxxxx 기반 멀티 클러스터 관리, AI 모델 배포/서빙, 웹 콘솔을 갖춘 **첫 공개 버전**입니다.

### Highlights

- **멀티 클러스터 배포** — xxxxxxx Push 모드로 여러 클러스터에 모델을 분산 배포
- **AI 모델 서빙 E2E** — 웹 콘솔에서 모델 선택 → 배포 → inference 호출까지 한 번에
- **서빙 런타임** — Ollama, vLLM, llama.cpp 등 5종 지원, GPU/CPU 자동 선택
- **웹 콘솔** — 대시보드, 클러스터, 정책, 배포, 인스턴스, 로그, 모델, 설정 (9개 페이지)
- **JWT 인증** — HttpOnly Cookie 기반 로그인, Access Token 자동 갱신
- **원커맨드 설치** — `./scripts/install.sh --registry-token <PAT>`로 H-MAS 전체 설치

### Quick Start

```bash
# 자동 설치 (xxxxxxx + H-MAS)
./scripts/install.sh --registry-token <GHCR-PAT>

# 접속
# 웹 콘솔: http://localhost:30000
# API:     http://localhost:30080
```
